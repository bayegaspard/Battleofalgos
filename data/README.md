# Data Sources

This directory contains all data sources used in the CyberSOCEval benchmark.

## Organization

- `crowdstrike-reports/`: CrowdStrike Intel Reports, Global Threat Reports, and other publications
- `hybrid-analysis/`: JSON data from Hybrid Analysis
