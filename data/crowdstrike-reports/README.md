# CrowdStrike Reports

This directory contains CrowdStrike reports used in the CyberSOCEval benchmark.

## Contents

This directory includes various types of CrowdStrike publications:
- Global Threat Reports
- Intelligence Reports
- Blog Posts

## File Format

Reports are provided in PDF format.
