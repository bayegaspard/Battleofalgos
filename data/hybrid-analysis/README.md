# Hybrid Analysis Data

This directory contains Hybrid Analysis data used in the CyberSOCEval benchmark.

## Contents

This directory includes JSON files containing analysis results from the Hybrid Analysis platform, providing detailed information about malware samples and their behavior.

## File Format

Data is provided in JSON format, with each file representing a single sample analysis.
